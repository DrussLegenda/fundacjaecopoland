# Fundacja Eco Poland

Statyczna strona Fundacji Eco Poland przygotowana do wdrozenia przez GitHub i Hostinger.

## Pliki publikowane

- `index.html`
- `styles.css`
- `app.js`
- `favicon.svg`
- `assets/`

## Hostinger

W panelu Hostinger wybierz wdrozenie z repozytorium GitHub, branch `main`, a jako katalog publikacji ustaw katalog glowny strony (`public_html` albo puste `Install Path`, jesli Hostinger sam publikuje do `public_html`).

Po kolejnych zmianach wystarczy wypchnac nowy commit na GitHub, a Hostinger pobierze aktualna wersje strony.
